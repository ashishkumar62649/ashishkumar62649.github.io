# Portfolio Website Notes

This project is a personal portfolio website for Ashish Kumar.

The landing page presents two directions:

- Design: the white side of the split experience.
- Coding: the black side of the split experience.

The site will eventually contain two portfolio modes:

- A design-focused website.
- A coding-focused website.

Typography notes:

- Design side: Georgia-style serif typography.
- Coding side large/title words: JetBrains Mono.
- Coding side smaller sentences and code-like text: IBM Plex Mono.

The landing interaction uses a split-screen selector where the design and coding sides respond to cursor movement.
