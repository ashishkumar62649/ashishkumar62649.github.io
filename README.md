# ashishkumar62649.github.io

Personal portfolio website of Ashish Kumar.

A clean, futuristic portfolio built with HTML, CSS, and JavaScript. Features a dark space theme with glassmorphism, responsive design, and dark/light mode toggle.

## Live Site

[https://ashishkumar62649.github.io](https://ashishkumar62649.github.io)

## Structure

- `index.html` - Home page with hero, projects preview, roadmap
- `projects.html` - All projects, coming soon, case studies
- `writing.html` - Technical writing and blog posts
- `learning.html` - Learning journey and resources
- `research-notes.html` - Research reading and technical notes
- `resume.html` - Resume (coming soon)
- `contact.html` - Contact information
- `css/style.css` - All styles
- `js/script.js` - Theme toggle, animations, mobile menu

## Features

- Dark mode (default) and light mode
- Responsive design for mobile, tablet, desktop
- Glassmorphism effects
- Smooth scroll animations
- Star field background
- Project roadmap with status badges
- Clean, minimal, professional design

## Deployment

Push to `main` branch. GitHub Pages will deploy automatically.

Replace `your-email@example.com` and `https://linkedin.com/in/your-linkedin` with your actual contact information in all HTML files.

Add your profile photo at `assets/profile.jpg`.
